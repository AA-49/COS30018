## Automated Negotiation System

## Summary

This project is a simple automated negotiation platform for buying and selling cars.
The system uses different agents:

* **Broker Agent (KA)** – manages listings and connects buyers with dealers
* **Dealer Agents (DA)** – sell cars and negotiate prices
* **Buyer Agents (BA)** – search for cars and negotiate for the best price

## Features
* Dealers can list cars on the platform
* Buyers can search for cars based on their needs
* Broker matches buyers with suitable dealers
* Buyers and dealers can negotiate prices
* Basic automated negotiation supported

## How It Works
1. Dealers send car listings to the Broker
2. Buyers request cars from the Broker
3. Broker matches buyers with dealers
4. Buyers and dealers negotiate prices
5. Deal is completed if both agree
